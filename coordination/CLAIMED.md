## Permanently Shared Files (edit carefully, pull first)
These files are touched by many systems — extra caution when editing:
- `settings.gradle` — root project list
- `gradle.properties` — shared version constants (forge, mc, javafx, mapcoreVersion) AND the JDK 17 daemon pin
- `build.gradle` (root) — group/version constants
- `mapcore/src/main/java/io/fathereye/mapcore/api/**` — public API; any change here forces a mapcore version bump and breaks the parallel session's `map/` build
- `docs/mapcore-api-contract.md` — versioned contract; only edit when bumping mapcore version
- `docs/ipc-protocol.md` — bridge ↔ panel wire format; both sides must move together
- `bridge/src/main/java/io/fathereye/bridge/topic/Topics.java` — topic name constants
- `bridge/src/main/java/io/fathereye/bridge/util/Constants.java` — pipe prefix, bridge version, protocol version
- `panel/src/main/java/io/fathereye/panel/App.java` — wires every subsystem; high coupling
